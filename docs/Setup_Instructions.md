# Setup Instructions

## Development Environment

* **IDE:** IntelliJ IDEA
* **JDK Version:** OpenJDK 20

## Verify Java Installation

Open Command Prompt and run:

```bash
java -version
```

The output should display the installed Java version.

Check the Java compiler:

```bash
javac -version
```

The output should display the installed JDK version.

## Running the Project

1. Clone or download the repository.
2. Open the project in IntelliJ IDEA.
3. Ensure the Project SDK is set to **OpenJDK 20** (or a compatible JDK version).
4. Navigate to `Main.java`.
5. Run the application.

## Hello World Verification

To verify the Java installation, create and run the following program:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

If the output displays:

```
Hello World
```

then Java has been installed and configured successfully.
